O Que é Código Binário e Como Funciona

O código binário é um sistema de representação de dados que utiliza apenas dois dígitos: 0 e 1. Ele é a base de toda a tecnologia digital moderna. Cada bit representa um estado lógico, como ligado (1) ou desligado (0).

Como os Números Funcionam no Sistema Binário

Diferente do sistema decimal (base 10), o sistema binário trabalha na base 2. Exemplo de conversão:

  • Decimal 1 = Binário 0001
  • Decimal 2 = Binário 0010
  • Decimal 3 = Binário 0011
  • Decimal 4 = Binário 0100
  • Decimal 5 = Binário 0101

Exemplo de Texto em Binário

A palavra OI em código binário:

01001111 01001001
